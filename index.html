Binary Search Tree Visualizer

  • Utilice la entrada "Insertar" para añadir números al árbol.
  • Utiliza la entrada "Delay" para ralentizar o acelerar el proceso de inserción.
  • Mantenga pulsado el botón izquierdo del ratón y arrástrelo para mover la cámara.
  • Utiliza el botón "Insertar Random" para insertar un número aleatorio en el árbol.
  • Utiliza las teclas de flecha arriba & abajo para acercar/alejar el zoom.
